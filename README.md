# Gestor de Tareas

App móvil del Parcial 1 de Apps Móviles, ISTEA 2026.

## Opción elegida

**Gestor de tareas**: agregar tareas con título y un recordatorio que dispara una notificación local.

## Cómo correrlo

Requisitos: Node 22 o superior, y ***Expo Go*** instalado en un dispositivo (o un simulador iOS / emulador Android).

```bash
git clone https://github.com/LeandroGlassman/ISTEAAppMobile.git
cd ISTEAAppMobile
npm install
npx expo start
```

Después:

- Escanear el QR con la cámara del iPhone, abre en ***Expo Go***
- O tocar ***i*** en la terminal para iOS Simulator
- O tocar ***a*** para emulador de Android
